Overlaying PNG over PNG for free! 

The background will be base_path
The second image will be overlay_path
The result images will be output_path

See sample code below:
base_path = "png1.png"
overlay_path = "png2.png"
output_path = "png3.png"

The position of second image is adjustable [x,y]
The size of second image is adjustable [scale]

Heads up! 
No interface available so you cant see the second image position at all
Just guess the position. Run the progam as you want while adjusting the right position
